# Palombo Architectural Design Website

## Goal
Build a clean, minimal architecture portfolio website.

## Pages
- Home
- About
- Projects (portfolio)
- Services
- Contact

## Design Style
- Minimal
- White background
- Large images
- Clean typography

## Notes
- Focus on showcasing projects
- Keep layout simple and elegant

## Content and asset guidance

- Use real project photos from `/public/images/projects`
- Use real drawings and sketches from `/public/images/drawings`
- Do not use fake testimonials
- Do not use placeholder stock images
- The visual style should feel like a premium architecture studio:
  minimal, refined, spacious, and image-led
